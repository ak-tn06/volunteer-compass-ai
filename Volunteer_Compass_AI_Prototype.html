Volunteer Compass AI
Prototype built for a Claude Corps Fellow application. This is a demonstration only and is not affiliated with any real organization.

About this project

Why I built this

Volunteer coordinators often answer the same onboarding questions repeatedly. This project explores how AI can personalize volunteer orientation while reducing repetitive onboarding work, so staff can spend more time supporting volunteers and less time repeating logistics.

Project goal

Help first-time volunteers feel informed, confident, and prepared before their first shift.

Intended audience

  • First-time nonprofit volunteers
  • Volunteer coordinators

MVP scope

This prototype intentionally focuses on onboarding only. It does not include volunteer scheduling, volunteer management, accounts, or a database — keeping the scope narrow was a deliberate product decision, not an oversight. It also includes a demo mode (mock responses, no API key required) alongside a live mode that calls the real Claude API — useful for evaluating the product experience without any setup.
